# 📖 TheQuran

A simple and beautiful Quran web app built to make reading the Holy Quran easy and accessible for everyone. 🌙

🌐 Live Website: https://wajidengg.github.io/thequran/

---

## ✨ Purpose

This project was created to provide a clean, fast, and distraction-free Quran reading experience online. 🤍

---

## 🚀 Features

* 📚 Read the Quran online
* 📱 Mobile-friendly design
* ⚡ Fast and lightweight
* 🌙 Simple and clean interface

---

## 🛠️ Built With

* HTML
* CSS
* JavaScript
* GitHub Pages
